# Youper Challenge
Youper Challenge developed in Ionic 1

## QUICK START GUIDE
1. Clone the repository https://github.com/fellps/Challenge-API and follow your installation steps
2. Change the 'www/config.js' file to your local settings

###Then run:

```
$ npm install
$ bower install
$ ionic serve --lab
```

###Unit Tests

```
$ gulp test
```
